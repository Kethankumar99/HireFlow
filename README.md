# 🔍 DeepHire
**Deep Analysis. Smart Hiring.**

AI-powered recruitment platform that reads every resume fully and ranks candidates based on actual capability - not keyword matching.

## Features
- 📄 JD Upload (File/Paste)
- 📁 Bulk Resume Upload (PDF/DOCX/TXT/ZIP)
- 🤖 AI Analysis via Groq
- 📊 GitHub-Style Results Table
- 📄 Original PDF Viewer Modal
- 📑 Pagination
- 📥 CSV & Excel Export
